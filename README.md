Projeto desenvolvido para lidar com a gestão do tempo de estudos.
# CYCLE
## Descrição:
O protótipo foi criado para resolver um problema referente ao cotidiano pessoal, especificamente sobre a gestão do tempo de estudos.​ Essa problemática foi escolhida devido ao fato de que como estudante, criar um projeto que resolvesse isso contribuiria não só pro avanço na disciplina, mas também seria um grande aprendizado particular.​
